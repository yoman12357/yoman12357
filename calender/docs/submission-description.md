# Submission Description

This project is an enhanced version of a frontend engineering challenge to build an interactive wall calendar component.  
It uses Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, and date-fns to deliver a polished, responsive calendar experience.  
Core functionality includes range selection, notes with localStorage persistence, reminders with browser notifications, and a clean wall-calendar layout across desktop and mobile.  
Beyond the original brief, I added drag-to-select interaction, animated month transitions, theme switching, quick actions, sticky desktop notes, and strong accessibility support.  
The implementation emphasizes frontend engineering quality through thoughtful UX feedback, edge-case handling, responsive behavior, and production-level polish.  
Overall, the project reflects a product-minded approach to interactivity, usability, and maintainable interface architecture.
