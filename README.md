## Featured Projects

### Traffic Correlation Research Platform

Built a full-stack cybersecurity research platform for traffic correlation analysis on anonymous networks. It supports simulation, replay, PCAP, and live capture workflows, with a Python analysis engine and a real-time dashboard for concurrent research sessions.

**Tech:** `Python` `Node.js` `Express.js` `Next.js` `SQLite` `WebSockets`

### AeroNITK Website

Designed and deployed the first official website for AeroNITK, making events, projects, and team information easier to explore for 200+ members.

**Tech:** `React.js` `Vite` `Tailwind CSS`

### eBPF-XDP Network Slicing Firewall

Implemented a kernel-level firewall based on eBPF/XDP and network slicing concepts from IEEE 6G research. The system classifies live traffic at wire speed and applies in-kernel filtering using threat-intelligence-backed rules.

**Tech:** `C` `eBPF/XDP` `Python` `libbpf` `Mininet`

### Darknet Route Tracking

Ongoing security research focused on detecting malicious activity on the Tor network through path inference, anomaly analysis, and attacker behavior tracking.

**Tech:** `Python` `Rust` `Network Analysis`

### FinSight

A personal finance analyzer with a Guardian Mode that helps enforce daily and monthly spending discipline through real-time tracking and alerts.

**Tech:** `React` `Firebase` `Node.js`

### Secure Browser

A privacy-focused browser in development, built around safer defaults, stronger isolation, and reduced tracking exposure.

**Tech:** `Electron` `Rust` `C++`
